Portal de Accesibilidad Universal

Diseñado para que todas las personas, con o sin discapacidad, puedan acceder a la información y comunicarse sin barreras.

👁 Discapacidad visual 👂 Discapacidad auditiva 🤝 Comunicación no verbal ⌨ Navegación por teclado

Soporte para Discapacidad Visual

Lectores de pantalla

Esta página usa etiquetas ARIA avanzadas, compatible con NVDA, JAWS y VoiceOver. Todas las imágenes tienen texto alternativo descriptivo.

Opciones visuales

  • Alto contraste activable desde la barra superior
  • Tamaño de fuente ajustable (A+ / A-)
  • Foco visible con borde azul en todos los elementos interactivos
  • Estructura de encabezados clara y ordenada

Soporte para Discapacidad Auditiva

Contenido multimedia

Todo el contenido de vídeo incluye subtítulos descriptivos y transcripciones completas descargables. Las alertas del sistema son siempre visuales, nunca solo sonoras.

[Ejemplo: vídeo con intérprete de lengua de signos en esquina y subtítulos de alta visibilidad]

Diseño sin barreras auditivas

  • Iconografía intuitiva para cada acción
  • Alertas visuales con color y animación
  • Lenguaje sencillo (lectura fácil)
  • Sin información transmitida solo por sonido

Comunicación No Verbal — Pictogramas

Para personas con dificultades en el habla, pulsa cualquier pictograma para que el sistema lo anuncie en voz alta y muestre una notificación visual.

Comandos de Voz Disponibles

Pulsa el botón "Comando de voz" en la barra superior y di uno de estos comandos:

Di "contraste" → activa alto contraste
Di "más grande" → aumenta la letra
Di "más pequeño" → reduce la letra
Di "ayuda" → anuncia menú de ayuda
Di "detener" → para el audio
Di "inicio" → vuelve arriba

Navegación por Teclado

Puedes controlar toda la página sin ratón:

Tab Siguiente elemento
Shift+Tab Elemento anterior
Enter Activar botón
Esc Detener audio
Alt+C Alto contraste
Alt+V Comando de voz
Alt++ Letra más grande
Alt+- Letra más pequeña
