♿ AccesoUniversal ✨ IA

AccesoUniversal es una plataforma web de vanguardia diseñada para eliminar las barreras digitales. Utiliza la potencia de la Inteligencia Artificial (Google Gemini) para ofrecer una experiencia inclusiva a personas con discapacidades visuales, auditivas, cognitivas y de comunicación.

🚀 Características Principales

🧠 Inteligencia Artificial (Gemini API)

Asistente de Simplificación: Traduce textos complejos a lenguaje sencillo para personas con dificultades de comprensión.

Narración Natural (TTS): Genera voces humanas pausadas y claras, superando la frialdad de los lectores de pantalla tradicionales gracias al modelo gemini-2.5-flash-preview-tts.

Comandos de Voz Inteligentes: Entiende la intención del usuario mediante procesamiento de lenguaje natural (NLP). La IA interpreta frases como "ponlo más oscuro" o "haz la letra más grande" para ejecutar acciones de accesibilidad sin comandos rígidos.

🛠️ Herramientas de Accesibilidad Clásicas

Alto Contraste: Interfaz optimizada con esquemas de color de alta visibilidad para personas con baja visión o daltonismo.

Control de Tipografía: Sistema de ajuste dinámico del tamaño de fuente para mejorar la legibilidad.

Pictogramas de Comunicación: Panel de comunicación aumentativa con salida de voz para personas con dificultades en el habla.

Navegación por Teclado: Implementación completa de roles ARIA y gestión de focos para una navegación fluida sin necesidad de ratón.

🛠️ Tecnologías Utilizadas

Frontend: HTML5, CSS3 (Tailwind CSS).

IA/LLM: Google Gemini 2.5 Flash Preview.

Voz: Gemini TTS (Audio Modality) & Web Speech API.

Diseño: Responsive Design (Adaptable a móviles, tablets y ordenadores).

📦 Instalación y Uso

Clona el repositorio:

git clone [https://github.com/tu-usuario/acceso-universal-ia.git](https://github.com/tu-usuario/acceso-universal-ia.git)


Configura tu API Key:
Abre el archivo index.html y localiza la variable apiKey. Inserta tu clave de Google AI Studio (puedes obtenerla en Google AI Studio):

const apiKey = "TU_API_KEY_AQUÍ";


Ejecuta el proyecto:
Abre el archivo index.html en tu navegador. Se recomienda el uso de Google Chrome para garantizar la compatibilidad total con las funciones de reconocimiento de voz.
